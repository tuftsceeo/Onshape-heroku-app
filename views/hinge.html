Approach #1


We tried!

We used LINE, CIRCLE, TANGENT, TRIM, MIRROR, LOFT, EXTRUDE, DIMENSION, and TRANSFORM.

Step 1: Use the Line Tool to Start Sketching Half of the Hinge

Select the line tool and start drawing 3 sides of a normal rectangle. In other words, draw the top, the left side, and the bottom.

Step 2: Use the Circle Tool to Draw the Center of the Hinge

Within the same sketch, click out of the circle tool and draw two cocentric circles right next to the 3 lines that you just drew.

The circles should be on the open side of the "rectangle," or the 3 lines, that you drew.

Approximate the size of the circles by having the diameters be about 4 times the "width" of the incomplete rectangle.

Step 3: Use the Line Tool again to Connect the Circles and the Lines

Click on the line tool and draw a line from the right edge of the top line of the incomplete rectangle (point closest to the circles) to the edge of the inner circle. Just make this line straight along the same axis.

Then, from that same starting point, draw another line angled downwards. It doesn't really matter where you stop the line, so just click anywhere inside or outside the circle. This will make sense later on.

From the right edge of the bottom line of the incomplete rectangle (the point right below where you started drawing the last line), draw another angled line downwards and click anywhere. If it helps, try to make this line parallel to the last angled line you just drew.

Step 4: Use the Tangent Tool to Connect the Circles to the Lines

Still in the same sketch, click on the tangent tool. Select 2 entities at a time that you want to be tangent to each other.

First click on the first line angled downwards and click on the inner circle edge. The line will now be tangent to the inner circle.

Repeat this for the other line angled downwards. Select the line and then click on the outer circle edge.

Step 5: Use the Trim Tool to Remove Excess Sketch Lines

In the same sketch, select the trim tool to remove the sketch lines that you no longer need/want.

You want to end up with a sort of hook at one end of half the hinge (the circular side should be the hook). View the end of the video to see what we mean.

Optional Step: Use the Dimension Tool to Resize your Hinge

Click on the dimension tool if you would like to change some of the part sizes for your hinge.

Step 6: Use the Extrude Tool to Make your Hinge 3D

You can either stay in the sketch and click the extrude tool or you can exit out of the sketch, click the extrude tool, and then select the face that you just created.

Set the height of the hinge to whatever you want. In our case, we will set it to 2.5 inches.

Step 7: Use the Circle Tool to Make Nail Holes

Start a new sketch on the flat surface of the hinge. Select the circle tool and draw a small-medium circle, or however big you think the diameter of a nail might be with respect to the hinge.

Draw this circle approximately at the vertical halfway mark. Perhaps have it close to the edge of the hinge as well.

Step 8: Use the Line Tool to Create a Construction Line

Using the line tool in the same sketch, draw a line all the way across the hinge, passing through the center of the circle that you just draw. Try your best to have this line exactly in the middle of the hinge. You may use the dimension tool to help you, but we will just approximate.

Right click on the line that you just drew and select contruction. You have now created a line that you can use as an anchor, measure sketches from, and mirror sketches across. This will be useful for later steps.

Step 9: Use the Circle Tool to Create More Nail Holes

Sketch another circle above the contruction line that you just created.

Right click on this circle and select "construction" again if you see that you are still in construction mode (the dotted outlines).

Use the dimension tool to make sure that the two circles you have are the same size.

Step 10: Use the Mirror Tool to Replicate the Circle

In the same sketch, use the mirror tool to create the same circle that you just drew on the other side of the construction line.

Select the line over which you wish to mirror the circle, and then click on the circle itself.

Afterwards, you may delete the construction line and click the green check mark to finish this sketch.

Step 11: Use the Circle Tool to Create the Holes on the Other Side of the Hinge

On the other side of the hinge, use the circle tool to create slightly smaller circles with the same center as the circles on the first side you drew them on.

You can see your first circle sketches on the first side by having them visible (the eye icon in the left bar next to the sketch).

You can use the dimension tool to make them all the same size.

Step 12: Use the Loft Tool to Remove the Holes

Select the loft tool to finally remove the holes from the circle sketches that you made.

When using the loft tool, you want to select two entities/faces: one face lofts to the other.

Select the respective large and small circles to remove the holes through the flat surface of the hinge.

Do this for all three of the holes

Step 13: Use the Plane Tool and Mirror Tool to Create the Other Half of the Hinge

Select the plane tool and offset the new Right plane by however many inches. We kept it to 1 inch.

Use the mirror tool and mirror the entire hinge half that you just created across the new plane.

Step 14: Use the Corner Rectangle Tool to Sketch the Hinge Knuckles

In a new sketch, select the corner rectangle tool to draw the knuckles of the hinge. Work your way down the hinge, and make sure the rectangles on one half of the hinge don't interfere with those on the other half.

If you would like, you can use the dimension tool to make exact measurements.

Step 15: Use the Extrude Remove Tool to Finish Creating the Hinges

Select the extrude tool and make sure that it is set to "Remove" instead of "Add."

First extrude remove the front half of the hinge knuckles and make sure to select "merge with all" to remove all of the rectangles.

Repeat that same process but for the back half of the hinge knuckles.

Step 16: Use the Transform Tool to Move the Two Halves Together

Select the transform tool and use the displayed arrows to move the two halves of the hinge together.

Step 17: Use the Circle Tool and Extrude Tool to Create the Pin

In the middle where the two hingles come together, draw a circle with the circle tool.

Use the extrude tool to extrude the circle to create a cylinder, which will act as a pin for the hinge. Make this as tall as the hinge itself.
