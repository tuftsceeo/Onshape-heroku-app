Paperclip Approaches


Overview of Steps


1) Use the Line Tool to Start Sketching the Paperclip

2) Use the 3 Point Arc Tool to Sketch the Curve of the Paperclip

3) Finish the Sketch of the Paperclip with the Line Tool and 3 Point Arc Tool

4) Use the Plane Tool to Create a New Plane at an End of the Paperclip

5) Use the Circle Tool to Create the Cross-Section of the Paperclip

6) Use the Sweep Tool to Make the Paperclip 3D

*) Use the Appearance Panel to Give your Paperclip some Color

Approach #1


We tried!

We used ARC, LINE, PLANE, CIRCLE, and SWEEP.

Step 1: Use the Line Tool to Start Sketching the Paperclip

Start off by creating a line anywhere on any plane. Here, we chose the Top plane and chose to sketch some distance away from the origin. However, to make our lives easier for a later step, we chose to start the line on the same horizontal axis (looking at it from the Top plane view).

You can use the Dimension tool to make this line a specific length. This is one side of the paperclip, and we put 1.5 inches.

Step 2: Use the 3 Point Arc Tool to Sketch the Curve of the Paperclip

Within the same sketch, click out of the Line tool and select the 3 Point Arc tool. We chose to make the paperclip as wide as the space between the initial line sketch and the origin.

With the 3 Point Arc tool, click once on the endpoint of the initial line sketch that lies on the same axis as the origin, then click again on the origin point. The last click determines the radius; make the radius the same all-around by clicking where the radius is "locked."

Step 3: Finish the Sketch of the Paperclip with the Line Tool and 3 Point Arc Tool

Essentially repeat the first two steps to finish off the paperclip as best as you can.

Remember what a paperclip looks like when you complete this step: it looks like an oval-shaped swirl.

Step 4: Use the Plane Tool to Create a New Plane at an End of the Paperclip

Click on the plane tool to create a new Front plane, or the plane that is perpendicular to your paperclip sketch. Use the Top view (or whichever view your paperclip is on) to make it easier to see.

Remember the first step, which was to draw a line. Try to offset your plane to the other end of that line (essentially where the paperclip starts). Also remember that we dimensioned it in the first step. This is where dimensioning it makes it easier. Simply offset the plane to however long you made that line. In our case, we will offset it by 1.5 inches since that is how long we made the line in Step 1.

Step 5: Use the Circle Tool to Create the Cross-Section of the Paperclip

On the new plane that you just made, use the circle tool to sketch a small circle at the end of the first line sketch that you drew.

Remember to make the circle relatively small since this circle is essentially the thickness of the paperclip.

Step 6: Use the Sweep Tool to Make the Paperclip 3D

Click on the sweep tool, which will prompt you to choose a face and a sweep path. Select the circle as the face. Then select the lines and the arcs that you drew as the sweep path: click on each part of it and you will see the sweep function working.

Click the green check mark once you have finished.

Optional Step: Use the Appearance Panel to Give your Paperclip some Color

Either right click on your new paperclip or go to the right and click on the Appearance Panel.

Use the Palette or the Mixer to choose a color.
